# KeYanGua-Archive 🔍

> 中国高校教授学术事件历史存档项目

## 📋 项目简介

本仓库收集、整理和存档中国高校教授相关的公开举报事件、师风师德及各类"吃瓜"事件。

科研人员常常被赋予过高的社会期待，但现实往往相去甚远。本项目通过客观记录这些真实发生的事件，帮助大家看清这个群体的真实样貌——他们也是普通人。

不指望这些记录能改变什么，只是如实保存正在发生的事情。

## 🎯 项目目的

- **打破神话**：让科研人员和公众认清这个群体的真实面貌，不再盲目崇拜
- **历史存档**：保存可能被删除或遗忘的公开信息，留下真实记录
- **信息整合**：将分散的信息集中整理，便于查阅和传播
- **现实认知**：直面学术圈的种种乱象，不抱不切实际的幻想

## 📝 收录标准

本项目收录的事件满足以下条件：

1. **公开性**：事件已在公开渠道（社交媒体、新闻媒体等）传播
2. **相关性**：涉及中国高校教授或研究人员
3. **重要性**：具有一定社会影响或学术意义
4. **真实性**：基于可验证的信息来源

### 收录范围

- 学术不端行为（抄袭、造假、剽窃、数据篡改等）
- 师德师风问题（骚扰、性侵、压榨学生等）
- 学术圈乱象（学阀、裙带关系、利益输送等）
- 举报投诉事件
- 各类"吃瓜"事件
- 其他值得记录的真实事件

**注**：不是所有教授都有问题，但有问题的确实不少。本项目只记录已经公开的事实。


## ⚖️ 免责声明
